# Pendahuluan
 
_Prompt_ adalah jenis masukan yang diberikan kepada kecerdasan buatan untuk mendapatkan respon yang diinginkan. 
_Prompt Engineering_ merupakan disiplin baru untuk dapat membangun dan melakukan optimasi terhadap prompt sehingga dapat menghasilkan keluaran yang efesien dan sesuai dengan tujuannya.

Prompt Engineering juga meliputi penggunaan prompt untuk melihat efektifitas dan akurasi sebuah model Large Language Modelling (LLM). 
Panduan ini akan berisi bagaimana dasar menggunakan prompt, teknik yang dapat digunakan untuk melakukan efesiensi, serta contoh-contoh baik teknis maupun praktis seperti penggunaannya dalam menjawab soal atau membuat sebuah cerita.

Dalam panduan ini, prompt akan sering disebut sebagai perintah (sebagai penerjemahan dari prompt) dan pengujian dilakukan dengan menggunakan `ChatGPT-4o mini` baik menggunakan chatbot di `chatgpt.com` atau `playground` yang disediakan dalam platform OpenAI.
